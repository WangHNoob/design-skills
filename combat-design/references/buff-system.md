# Buff 与状态系统设计参考

## Buff 分类

| 类型 | 说明 | 叠加上限 | 可驱散 |
|------|------|----------|--------|
| 增益Buff | 提升属性（ATK/DEF/SPD等） | 3层 | 是 |
| 减益Debuff | 降低属性 | 不可叠加 | 是 |
| 护盾 | 吸收伤害 | 1层 | 否 |
| 控制 | 眩晕/冰冻/沉默等 | 不可叠加 | 部分 |
| 光环 | 持续生效的范围效果 | 1层 | 否 |

## 状态判定顺序

```
战斗开始 → 被动生效 → 行动判定(SPD排序) → Buff过期检测 →
行动执行 → 伤害计算 → 死亡判定 → 下一回合
```

## 常见状态效果

| 状态 | 效果 | 持续回合 | 免疫条件 |
|------|------|----------|----------|
| 眩晕 | 跳过行动 | 1 | 控制免疫 |
| 冰冻 | 跳过行动+受伤增加 | 1-2 | 控制免疫 |
| 沉默 | 无法使用技能 | 2 | 控制免疫 |
| 中毒 | 每回合扣血 | 3 | 毒免疫 |
| 灼烧 | 每回合扣血+降防 | 2 | 火免疫 |
| 无敌 | 不受伤害 | 1 | 不可驱散 |
