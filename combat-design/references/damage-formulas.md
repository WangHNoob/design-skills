# 伤害公式参考

## 基础伤害公式

```
基础伤害 = ATK_attacker * skillMultiplier
减伤后 = 基础伤害 * (1 - DEF_defender / (DEF_defender + K))
其中 K 为防御常数，建议值 1000
```

## 常见伤害类型

| 类型 | 判定属性 | 特殊规则 |
|------|----------|----------|
| 物理伤害 | ATK vs DEF | 可被格挡 |
| 法术伤害 | MATK vs MDEF | 无视格挡 |
| 真实伤害 | 无 | 无视所有防御 |
| 百分比伤害 | 目标当前HP | 有上限 |
| DOT | 施法者ATK | 每回合触发 |

## 暴击与闪避

```
暴击判定: if random(0,1) < CRIT - CRT_resist → 暴击
暴击伤害 = 基础伤害 * (1 + CDMG)
命中判定: 命中率 = 1 - max(0, DODGE - HIT * 0.01)
```

## 属性克制

```
克制系数: 1.3 (克制) / 1.0 (普通) / 0.7 (被克制)
```
